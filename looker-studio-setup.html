GCP Aviation Lakehouse Platform  ·  Project: gcp-lakehouseproject  ·  Dataset: aviation_analytics

Step 1 — Open Looker Studio

1 Click the button below to open Looker Studio, then sign in with your Google account (same account that owns the GCP project).

Open Looker Studio → + Create Blank Report

Step 2 — Add Data Sources (one per view)

Click each link below to open the BigQuery connector pre-filled for that view. After adding the first source, add others via Resource → Manage added data sources → Add a data source.

Step 3 — Build the Charts

Add the following charts to your report. Use Insert → Chart for each one.

Scorecard ×3

KPI Banner

Data source: bi_airline_performance_v
Metrics: SUM(total_flights), AVG(avg_delay_min), AVG(delayed_pct)

Bar Chart

Delay by Airline

Data source: bi_airline_performance_v
Dimension: airline   Metric: avg_delay_min
Sort: descending by avg_delay_min

Time Series

Daily Delay Trend

Data source: bi_daily_delays_v
Dimension: flight_date   Metric: avg_delay_min

Table

Route Performance Table

Data source: bi_route_performance_v
Dimensions: route   Metrics: total_flights, avg_delay_min, delayed_pct, weather_impact_pct

Pie / Donut

Weather vs Non-Weather Delays

Data source: bi_airline_performance_v
Dimension: airline   Metric: weather_impact_pct

Scorecard

Pipeline Freshness

Data source: bi_pipeline_refresh_v
Metric: generated_ts (latest refresh timestamp)

Step 4 — Add a Date Range Filter

4 Go to Insert → Date range control and set the default range to Last 30 days. Apply it to all charts using bi_daily_delays_v.

💡 Tip: Hold Shift and click multiple charts to apply a single date control to all of them at once via Set date range.

Step 5 — Share the Report

5 Click Share in the top right → set to "Anyone with the link can view" → copy the URL. The report auto-refreshes from BigQuery on each open.

⚠️ Make sure the viewer's Google account has roles/bigquery.dataViewer on the aviation_analytics dataset, or set the data source to use owner credentials.
